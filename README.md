A CLI chatbot with persistent memory, built entirely with core Python:

- **OOP** — `ChatManager` class handles all state
- **JSON persistence** — conversation history saved/loaded between sessions
- **Error handling** — graceful fallback on missing files or bad data
- **Timestamps** — every message tagged with time
